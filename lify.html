jim's recent projects

lify

a shopping experience to support independent businesses

project aims:

  1. provide an alternative to mainstream shopping solutions
  2. develop a low cost service to allow small local shops to engage with an online customer base
  3. create a scaleable brand that can expand geographically to new locations

approaches and outcomes

deployed using NodeJS as a REST api, React and MongoDB on a free VPS

implemented payment systems with stripe to handle low cost transactions

integrated google maps api to provide simplified routing and delivery systems to customers

deployed to github for version control.
